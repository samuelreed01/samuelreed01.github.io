An ERP (Enterprise resource planning) Progressive Web App designed to be run on local networks, handling inventories, billing, and customer information.

I worked on this project as an async remote full-stack developer along with a remote international team. My role was to close issues and implement features using the customer's own library of custom components. This was different from my usual role taking full ownership of a codebase. The lead developer also uses JavaScript instead of TypeScript. This taught me new lessons in flexibility and working together in a team.
