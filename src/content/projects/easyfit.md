A progressive web app giving exercise reminders and guided workouts for sedentary office workers.

I worked on this project as an async remote full-stack developer along with a remote international team. My role was to close issues and implement features using the customer's own library of custom components. This was different from my usual role taking full ownership of a codebase. The lead developer also uses JavaScript instead of my preferred language of TypeScript. This taught me new lessons in flexibility and working together in a team.

Apart from working on existing features, I added the push notification system to allow live updates via VAPID-identified messages and the Push API.
